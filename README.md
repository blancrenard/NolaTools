# NolaTools

Unity Editor 拡張ツール - VRChat アバター制作を支援するツール

### FurMaskGenerator
VRChat アバター用のファーマスク自動生成ツール

- **機能**: アバターのファーマスク自動生成
- **使用方法**: Unity Editor で `Tools > NolaTools > FurMaskGenerator` を開く
- **要件**: Unity 2022.3 LTS 以降、VRChat SDK3
